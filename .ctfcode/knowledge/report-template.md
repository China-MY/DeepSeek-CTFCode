# 渗透测试报告严格格式模板

## 文档命名规范
- 报告路径：`/root/ATKING/报告md/{项目名}-渗透测试报告.md`
- 利用脚本：`/root/ATKING/exploit_*.py`

## 🚨 报告格式 — 严格遵循

**每条漏洞记录必须使用以下固定格式，字段名和顺序不可改变：**

```
项目	{项目名称}
漏洞详情	{漏洞名称}
漏洞风险等级	严重/高危/中危/低危/信息
漏洞地址	{URL}
漏洞描述与原理	{漏洞的技术描述和原理说明，包括后端技术栈版本}
验证步骤	见附件 Python 验证脚本
利用方式与影响	{攻击者视角的利用方式和业务影响说明}
临时缓解措施	{临时的防御/缓解方案}
永久修复建议	{永久性的修复建议}
```

### 格式铁律
1. 字段名和值之间使用 **Tab（\t）** 分隔，不是空格
2. **字段值末尾禁止出现 `|`（管道符）**。每个字段只包含纯文本，无任何装饰符号
3. 每条漏洞记录之间**空一行**，字段之间**不空行**
4. **严禁**使用 Markdown 表格（`|...|`）、列表（`-`）、标题（`#`）或其他格式标记
5. **所有字段必须非空**，不允许"暂无"、"待补充"、"无"等占位
6. `验证步骤`字段**固定写**"见附件 Python 验证脚本"
7. `利用方式与影响`必须从**攻击者视角**写
8. `漏洞风险等级`严格使用：严重/高危/中危/低危/信息

### 正确示例（Tab分隔，纯文本，无管道符）

```
项目	生物多样性智能监测平台
漏洞详情	GeoServer WFS/WMS 未授权数据访问
漏洞风险等级	高危
漏洞地址	http://112.24.99.137:8028/geoserver/
漏洞描述与原理	GeoServer 的 WFS/WMS/WCS/OGC 服务完全未配置任何认证机制。后端为 Jetty 9.4.57。任何未授权用户可通过标准 OGC 请求导出所有地理空间数据。
验证步骤	见附件 Python 验证脚本
利用方式与影响	攻击者可无需任何认证：导出全部104个自定义图层、312534条地理空间记录；获取国家重点保护野生动物的精确栖息地位置；利用WFS Transaction能力修改或删除服务器上的数据。
临时缓解措施	在nginx层添加访问控制规则，限制/geoserver/路径仅允许可信IP访问。
永久修复建议	1.配置统一认证拦截器；2.限制内网访问；3.升级至最新版本。
```

### 错误示例（严禁）

```
项目	生物多样性智能监测平台 |          ← 末尾的 | 是非法的！
漏洞详情	GeoServer WFS/WMS 未授权数据访问 |  ← 严禁管道符
漏洞风险等级	高危 |

| 漏洞名称 | 等级 | URL |              ← Markdown表格格式也是非法的！
```

## 危害提升复核

**漏洞定级前必须复核以下清单。发现有未尝试的提升路径时，handoff 退回 Exploit Agent：**

| 原始发现 | 必须尝试的提升路径 |
|----------|------------------|
| SQL注入 | xp_cmdshell / INTO OUTFILE / LOAD_FILE |
| 文件读取 | 日志投毒RCE / 密钥复用 / session伪造 |
| XSS | 凭据窃取 / CSRF配合 |
| 目录遍历 | 源码泄露 / 配置泄露 / 备份文件 |
| 弱口令 | sudo提权 / SSH密钥提取 / 数据库写Shell |
| 文件上传 | WebShell → RCE |

## CVSS 3.1 评分
| 等级 | 分数 | 条件 |
|------|------|------|
| 严重 | 9.0-10.0 | 远程无需认证RCE/数据全量泄露 |
| 高危 | 7.0-8.9 | 越权敏感数据/SQL注入/认证绕过 |
| 中危 | 4.0-6.9 | XSS/目录遍历/信息泄露 |
| 低危 | 0.1-3.9 | 文本信息泄露 |
| 信息 | 0.0 | 技术栈指纹 |

## 可信度
- A: 确认 + Python Exploit 利用成功 + 提权尝试完成
- B: 高度疑似（多种工具交叉验证）
- C: 可疑（单一来源）
- D: 误报（无法复现，不可删除，标注原因）

## 证据要求
每个漏洞必须有：
1. Python Exploit 脚本（`/root/ATKING/exploit_*.py`）
2. 命令回显（id → uid=0, cat /etc/passwd 等）
3. 文件内容（/.env、数据库配置等）
4. 提升尝试记录（**无论成功失败**）
